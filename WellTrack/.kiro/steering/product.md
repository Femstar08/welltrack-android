# WellTrack Product Overview

WellTrack is a comprehensive health and wellness Android application that integrates nutrition tracking, meal planning, and health data management. The app provides users with:

- **Recipe Management**: Import, store, and organize recipes with OCR scanning capabilities
- **Meal Planning**: Automated weekly meal plan generation with calendar view
- **Health Integration**: Connects with Health Connect, Garmin, and Samsung Health
- **Authentication**: Secure user authentication via Supabase
- **Accessibility**: Full WCAG 2.1 AA compliance with TalkBack support
- **Data Security**: Encrypted storage and biometric authentication

The application targets Android API 26+ with modern Material 3 design and Jetpack Compose UI.
